# 領域牌可視化ツール

[雀魂](https://mahjongsoul.com) イベント青雲の志向けの補助ツールです。
ドラ表示牌を入力すると、「神の領域+」で手牌に使える牌を可視化します。

Github Pages にデプロイしています。
以下の URL から使用できます。

https://sankantsu.github.io/seiunn-god-region-visualizer/

## 技術構成

React + Vite を使っています。

## 牌画

[麻雀王国](https://mj-king.net/sozai/) よりお借りしています。
